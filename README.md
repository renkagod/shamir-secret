# Shamir Secret

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Консольная утилита для схемы разделения секрета Шамира (k-of-n): разбивает текст на `n` долей и восстанавливает его по любым `k` долям.

## Особенности

- Полиномиальная схема над простым полем GF(p), p = 2^521 − 1
- Порог `k` из `n` (для восстановления нужно минимум `k` долей)
- Случайные коэффициенты через `secrets` (stdlib)
- Без внешних зависимостей

## Требования

- Python 3.10+

## Запуск

```bash
python main.py
```

### Меню

| Пункт | Действие |
| ----- | -------- |
| `1` | Разделить секрет — ввести текст, число частей `n` и порог `k` |
| `2` | Собрать секрет — ввести `k` долей в формате `номер значение` |
| `0` | Выход |

### Пример

Разделение «пароль» на 5 частей с порогом 3 — для восстановления достаточно любых трёх долей из пяти.

## Ограничения

- Секрет кодируется как целое из байт UTF-8; для выбранного `p` практичная длина текста — порядка ~60 символов
- Учебная реализация; для продакшена используйте проверенные криптобиблиотеки
